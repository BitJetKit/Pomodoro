# Pomodoro Timer
On the basis of the freeCodeCamp app tutorial, part of InspiredWebDev,
this app is a JavaScript app using the Pomodoro Technique.
I am using the freeCodeCamp code.
The purposes for making this app are review, learning, and overhead reduction
as a JavaScript app requires less space than a Java app
and JavaScript can be loaded onto an Android IDE easily enough. This app
shall have the original tutorial code, and I shall add a browser notification
option, and this shall enable a productive work environment because it is
better than awkwardly checking the browser tab, irregularly. I shall add documentation because recording the guidance steps a developer uses is vital,
and documentation is one of my talents. I shall also do the artwork: using 3D Paint, I can reduce overhead and eliminate artwork usage redundancy. Against being a plagiarist, I would not want Google's logo on my app: for example. Feel free to modify my code, a courtesy of my business, Bit Jet Kit.
